# Manson Digital Forensics Investigation

## Overview

This repository contains a digital forensic case study involving suspected prohibited firearm buying and selling activity on a Windows 98 computer.

The investigation focused on distinguishing casual firearm-related browsing from deliberate sales activity. Browser records, downloaded images, locally stored documents, email artifacts, user directories, and timestamps were correlated to assess user behavior and attribution.

## Investigation Objectives

The examination focused on determining whether:

- Firearm-related activity represented casual browsing or organized sales behavior
- Images were deliberately downloaded rather than automatically cached
- Sales and planning documents were intentionally created or stored
- Email communications supported firearm-sales activity
- Relevant activity could be associated with the Jake user profile
- Windows 98 limitations affected definitive user attribution

## Tools Used

- AccessData Forensic Toolkit 8.2
- Avast Antivirus
- MD5 hash verification

## Key Skills Demonstrated

- Legacy Windows forensics
- FAT32 file-system analysis
- Index.dat analysis
- Browser-history reconstruction
- Downloaded-file analysis
- Email-artifact analysis
- Deleted-file recovery
- User attribution
- Timeline correlation
- Formal forensic reporting
